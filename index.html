How to paste into Gmail
  1. Open Gmail → ⚙️ Settings → See all settings.
  2. Scroll to Signature and create a new signature.
  3. Click inside the signature box and paste.
  4. Scroll down and click Save Changes.

Tip: If paste looks wrong, try pasting once into a blank Google Doc, then copy from the Doc into Gmail.
